# 本人的算法竞赛题解
这是我在学习信息竞赛过程中做过的，比较有思考价值的题目的简单分析。
这些题目有些是我仅凭自己思考就做出来的，但大部分是看过别人的题解后写出来的。
虽然本人水平很差，但是确有一些自己的思考，又因网上大量题解质量低下，故在此发表。
其中我大部分探讨的是思路，代码由C/C++风格的伪代码呈现。
